Free Code Camp - Image Search Abstraction Layer Microservice

User stories:

  • 1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • 2) I can paginate through the responses by adding a ?offset=2 parameter to the URL.

Example usage:

https://resolutedreamer-fcc-imagesearchabstractionlayermicroservice.gomix.me/api/imagesearch/lolcats%20funny?offset=10
https://resolutedreamer-fcc-imagesearchabstractionlayermicroservice.gomix.me/api/latest/imagesearch/
